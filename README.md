# 🛡️ Smart Data Auditor 2026
Application d'audit et de nettoyage de données propulsée par l'IA.

## ✨ Fonctionnalités
- **Nettoyage Rapide :** Suppression de doublons et formatage avec Polars.
- **Analyse IA :** Posez des questions en langage naturel sur vos fichiers CSV.
- **Visualisation :** Graphiques automatiques pour comprendre vos données.

## 🚀 Installation
1. Clonez le dépôt
2. Installez les dépendances : `pip install -r requirements.txt`
3. Lancez l'app : `streamlit run app.py`
